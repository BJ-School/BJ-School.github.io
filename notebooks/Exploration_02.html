Part 2: Calculate Summary Statistics

The marketing team has determined that when choosing a cereal, consumers are most interested in calories, sugars, fiber, fat, and protein.

First, letâ€™s calcuate some summary statistics for these categories across the entire dataset. Weâ€™re particularly intrested in the mean, median, standard deviation, min, and max values.

There are multiple ways to accomplish this.

# Part 2: Enter your code below to calculate summary statistics for the
# calories, sugars, fiber, fat, and protein features.
def calculateSummaryStatistics(theData, fieldsOfInterest):
  summaryStatisticsMean = theData[fieldsOfInterest].mean()
  summaryStatisticsMedian = theData[fieldsOfInterest].median()
  summaryStatisticsMin = theData[fieldsOfInterest].min()
  summaryStatisticsMax = theData[fieldsOfInterest].max()
  summaryStatisticsStd = theData[fieldsOfInterest].std()

  # Concatenate
  summaryStatisticsMerged = pd.concat([summaryStatisticsMean, summaryStatisticsMedian, summaryStatisticsMin, summaryStatisticsMax, summaryStatisticsStd], axis=1)

  # Label the columns
  summaryStatisticsMerged.columns = ['Mean', 'Median', 'Min', 'Max', 'Std']

  return summaryStatisticsMerged

theFieldsOfInterest = ['calories', 'sugars', 'fiber', 'fat', 'protein']
theStatistics = calculateSummaryStatistics(theData, theFieldsOfInterest)

print(theStatistics)

Part 3: Transform Data

To make analysis easier, you want to convert the manufacturer codes used in the dataset to the manufacturer names.

First, display the count of each manufacturer code value used in the dataset (found in the mfr column).

Then, create a new column with the appropriate manufacturer name for each entry, using this mapping:

A = American Home Food Products
G = General Mills
K = Kelloggs
N = Nabisco
P = Post
Q = Quaker Oats
R = Ralston Purina

Note: While the tutorial linked above uses the replace function, using the map function instead can often be much faster and more memory efficient, especially for large datasets.

# Display the count of values for the manufacturer code ("mfr" column), then
# create a new column containing the appropriate manufacturer names.
def getManufacturer(theAbbreviation):
  manufacturerData = {
    'A': 'American Home Food Products',
    'G': 'General Mills',
    'K': 'Kelloggs',
    'N': 'Nabisco',
    'P': 'Post',
    'Q': 'Quaker Oats',
    'R': 'Ralston Purina'
  }
  return manufacturerData.get(theAbbreviation, '')

manufacturerCount = theData[['mfr']].value_counts()
print(manufacturerCount)

theMappedData = theData
theMappedData['manufacturer'] = theMappedData['mfr'].map(getManufacturer)

Part 5: Dietary Calculations

The marketing team has been able to arrange a partnership between the popular Weight Watchers diet brand and Kelloggs cereal.

The Weight Watchers system assigns a point value to each food, and participants in the program are allotted a certain number of points each day.

One recent formula for calculating points is:

(Calories * .0305) + (Fat * .275) + (Sugar * .12) - (Protein * .098)

With the final answer being rounded to the nearest integer.

First, add a new column with the Weight Watchers point calculation derived from the existing data.

Be sure to round the number to the nearest int and store the data as an int, not as a float with 0 decimals.

Then, select a subset of the data containing just cereals manufactured by Kelloggs.

Finally, calculate the same summary statistics for the points calculations as earlier (mean, median, standard deviation, min, and max).

def calculatePoints(theFood):
  return round((theFood.calories * .0305) + (theFood.fat * .275) + (theFood.sugars * .12) - (theFood.protein * .098))

theMappedData['weightWatchesPoints'] = theMappedData.apply(calculatePoints, axis=1)
onlyKelloggs = theMappedData[theMappedData['manufacturer'] == 'Kelloggs']
